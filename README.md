# docker-buildpack

This repository is the source for the Docker Hub image `renovate/buildpack`. Commits to `master` branch are automatically built and published.

The image is used as a centralized build source for:
- [x] `renovate/composer`
- [x] `renovate/go`
- [x] `renovate/gradle`
- [ ] `renovate/npm`
- [ ] `renovate/pipenv`
- [ ] `renovate/pnpm`
- [ ] `renovate/poetry`
- [ ] `renovate/ruby`
- [ ] `renovate/yarn`
